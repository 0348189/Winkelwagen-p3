# Autodealer_php

klant wil een website waarop hij zijn auto's kan verkopen en waarop klanten de auto's kunnen bekijken en kopen.
specifieke dingen moeten aanpas paar zijn en verwijderd kunnen worden.
de producten op de web pagina moet opgesplagen worden in een database.
de klant moet een account kunnen aanmaken en inloggen.
en de klant moet een auto kunnen kopen en een auto kunnen verkopen.
de auto's moeten op de web pagina worden weergegeven.
en in de database moet de auto worden opgeslagen.


je moet je eigen naam gebruiken.

ontwikkel je eigen functionaliteit in een aparte branch

iedereen meot git gebruiken met de command line zadat je het gebruik goed leer

voor dit project schrijven we niet meer alle user stories uit voor je ui de globale omschijvingen zul je de user strories met je team moeten maken. zorg dat de story;s niet groter zijn dan 8 punten.
